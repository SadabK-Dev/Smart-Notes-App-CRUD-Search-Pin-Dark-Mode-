# Smart-Notes-App-CRUD-Search-Pin-Dark-Mode-
A fully functional Notes App built using Vanilla JavaScript with features like adding, editing, deleting, searching, pinning notes, and dark mode support. The app uses LocalStorage for persistent data and follows a clean UI with responsive design.

✨ Features

- Add, edit, and delete notes
- Search notes in real-time
- Pin important notes (sorted at top)
- Dark mode with saved preference
- Responsive grid layout
- LocalStorage data persistence
- Clean and modern UI
